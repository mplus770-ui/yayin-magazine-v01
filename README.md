# YAYIN · מגזין יין ישראלי

גיליון 01 · 2026 — מגזין יין ישראלי דיגיטלי בעברית, עיצוב Le Monde Sépharde.

## תיאור

מגזין פליפבוק אינטראקטיבי עם אפקט הפיכת עמודים תלת-ממדי.  
בנוי כ-HTML/CSS/JS טהור, ללא תלויות חיצוניות.

## קבצים

| קובץ | תיאור |
|------|-------|
| `index.html` | מבנה המגזין — כל הספרדים, הניווט, ומנוע הפליפבוק |
| `style.css` | כל העיצוב — טיפוגרפיה, עמודים, אנימציות |
| `img/` | תמונות המגזין |

## פיצ'רים

- הפיכת עמודים תלת-ממדית (CSS 3D transform)
- ניווט מקלדת (חצים), מגע (swipe), וקליק אזורי
- זום + / – וגלגלת עכבר (Ctrl+Scroll)
- קפיצה של 10 עמודים קדימה/אחורה
- כפתור חזרה לשער
- תמיכה במסך מלא והדפסה
- RTL עברי מלא
